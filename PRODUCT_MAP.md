# Product Map

This is a product direction map, not a commitment list. Use it for context, prioritization, and avoiding dead-end architecture. Do not implement everything by default.

## Template Builder Core

- Multi-page template editor
- Section add/remove/reorder/hide/show
- Undo/redo
- Desktop/tablet/mobile preview
- Preview zoom controls
- Page management for standard ecommerce pages and custom pages
- Section presets
- Page presets
- Full template starters
- Template duplication
- Template version history

- Typography presets

- Section-level style controls
- Button style controls
- Background controls
- Spacing and layout controls
- Per-section responsive settings
- Global header/footer settings
- Brand kit for logo, favicon, fonts, and colors
